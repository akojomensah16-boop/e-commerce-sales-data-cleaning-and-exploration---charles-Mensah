# e-commerce-sales-data-cleaning-and-exploration---charles-Mensah
Data cleaning and exploratory data analysis (EDA) of an e-commerce sales dataset using Excel. The project includes data quality checks, descriptive statistics, outlier detection, and pivot table analysis.

OVERVIEW
This project focuses on the initial stages of the data analytics process using an e-commerce sales dataset.
This repository contains an e-commerce sales dataset with 1,200 customer transactions recorded between January 2023 and June 2025.

The primary objective was to clean the dataset, identify data quality issues, explore key statistical measures, detect outliers, and summarize insights using pivot tables.

Objectives

The project was completed to:
 • Assess data quality
 • Identify missing values and inconsistencies
 • Detect outliers
 • Generate descriptive statistics
 • Explore sales trends
 • Summarize findings using pivot tables

Data Cleaning Process

The following cleaning tasks were performed:

1. Missing Value Checks
 • Reviewed all columns for blank or null values.
 • Determined appropriate treatment methods where necessary.

2. Duplicate Detection
 • Checked for duplicate records.
 • Removed duplicates where applicable.

3. Data Type Validation
 • Verified dates, numerical fields, and text fields.
 • Corrected formatting inconsistencies.

4. Consistency Checks
 • Reviewed categorical values for spelling variations and inconsistencies.

Exploratory Data Analysis (EDA)

Descriptive Statistics

Key statistical measures calculated include:
 • Count
 • Mean
 • Median
 • Minimum
 • Maximum

These statistics provided an overview of the dataset distribution and central tendencies.

Outlier Detection

Outliers were identified using the:
 • Interquartile Range (IQR) Method

Steps
 1. Calculate Q1 and Q3
 2. Compute IQR
 3. Determine lower and upper bounds
 4. Flag observations outside the acceptable range

Pivot Table Analysis

Pivot tables were created to summarize:
Average Unit price by Product
 • Identify top-priced products.

Sales by Product
 • Identify top-performing products.

Sales by Payment Method
 • Compare customer payment preferences.

Sales by Order Status
 • Analyze completed, pending, and cancelled orders.

Revenue Trends
 • Evaluate sales performance across periods.
